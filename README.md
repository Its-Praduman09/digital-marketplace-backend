# 🛒 Digital Marketplace Backend API

A professional-grade, scalable backend for a digital product store (mini-Gumroad/Udemy clone). This project demonstrates modern Node.js best practices, focusing on performance, type safety, and clean architecture.

## 🚀 Tech Stack
- **Runtime:** Node.js (TypeScript)
- **Framework:** [Fastify](https://www.fastify.io/) (High performance, low overhead)
- **Database:** PostgreSQL
- **ORM:** [Drizzle ORM](https://orm.drizzle.team/)
- **Validation:** [Zod](https://zod.dev/)
- **Authentication:** JWT + Bcrypt
- **Storage:** AWS S3 (Digital Asset Management)
- **Payments:** Razorpay Integration

## ✨ Key Features
- **Secure Auth:** Email/Password & OTP-based authentication.
- **Product Management:** CRUD for digital items with secure S3 file uploads.
- **Order System:** Transactional flow with payment gateway webhooks.
- **Data Integrity:** Strict schema validation using Zod and Drizzle.
- **Scalability:** Modular Controller-Service-Repository architecture.
